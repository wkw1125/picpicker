# 皮卡比 PicPicker 需求规格

本文件整理了皮卡比（PicPicker）的详细需求与交互设计，最初来源于 `evideo-image-utils/docs/AI辅助编程.md` 中的相关章节。

## 实现标记功能

1. 比较文件夹1、比较文件夹2的支持选择标记功能
2. 比较文件夹1、比较文件夹2的文件名旁放置一个标志位，可以使用一个绿色的打勾图案表示该图片被选择，默认未选中
3. 维护每个图片的选中状态数据，在比较文件夹1/2的预览图下方，按“已选择 n/m percent%”格式显示，其中 n 为已标记的图片数，m 为该文件夹下图片总数，percent 为选中占比（保留整数位）
4. 键盘左、右 可以分别切换比较文件夹1/2 图片的选中状态，选中状态时更新选中标志与状态数据

### 顶部信息显示调整

1. 首行文件夹路径使用全路径，不含文件名  
2. 去掉第二行的文件绝对路径显示  
3. 将尺寸信息放到文件名右侧，显示格式：(宽x高)  
4. 缩小行间距，尽量留出空间给图片预览区域

## 实现标记信息导出功能

将原图、图1、图2，选中信息导出为 csv 格式，按以下格式：

|-|原图|图1|图1选中态|图2|图2选中态|
|-|-|-|-|-|-|
|目录|原图目录|比较文件夹1目录|-|比较文件夹2目录|-|
|文件名|001.png|001.png|1|001.png|0|

选中态：若图片被选中为 1，否则为 0。

## 增加遮罩显示功能

图1、图2有对应的遮罩图片，可以在预览图位置对图片/遮罩进行切换。

1. 在原图、图1、图2 文件夹选择按钮下一行，依次增加「切换遮罩/图片」、「选择图1遮罩文件夹」、「选择图2遮罩文件夹」  
2. 在遮罩文件夹选定后，默认不显示。当点击「切换遮罩/图片」按钮时，图1、图2 切换为对应的遮罩图片。图1 遮罩与图1 绑定，图2 遮罩与图2 绑定，按相同的文件序号显示  
3. 原图没有遮罩相关的显示逻辑，始终不动

## 导出标记图片

在导出标记按钮右侧放置「导出标记图片」，按以下规则打包为 zip：

1. 图1 若被选中，打包图1 到 `/img1`；如果有对应遮罩图，打包至 `/mask1`；未被选中的无需打包  
2. 图2 处理逻辑与图1 一样，分别打包到 `/img2` 与 `/mask2`  
3. 图1 或图2 被选中，则对应的原图打包至 `/img`；图1、图2 同时被选中时，只需要打包一份原图  
4. 将 `/img` `/img1` `/mask1` `/img2` `/mask2` 打包为一个 zip 包进行保存，默认保存至下载目录  
5. 未选中任何图时，不进行打包

## 实现标记文件打开功能

基于现有的 csv 导出信息，实现导入功能：

1. 「文件」菜单增加「从标记文件打开」，作为该功能入口  
2. 选择一个 `.csv` 文件，按现有 csv 导出规则解析该文件，获得原图、图1、图2 的文件夹地址，以及该文件夹下每个文件对应的标记状态，根据这些信息实现界面的刷新（包括文件路径、选中状态、统计信息）  
3. 若从 csv 读取的三个文件夹图片数量与实际路径读取到的文件数量不一致，给出不一致警告弹窗提示  
4. 若从 csv 读取的三个文件夹图片文件顺序与实际路径读取到的文件顺序不一致，给出不一致警告弹窗提示

## 实现放大镜功能

基于你对「放大镜」功能的理解实现：

1. 「显示」菜单下增加「放大镜」功能，表示是否开启放大镜  
2. 鼠标位于三个预览图的任意位置时，三个预览图同步显示各自的放大镜；各自放大镜的放大位置相对图片位置保持一致同步。鼠标不在预览图内时，不显示放大镜  
3. 鼠标所在预览图的放大镜（称为主镜）显示在鼠标附近，其他两个预览图的放大镜同步主镜相对位置  
4. 放大镜窗口大小为 200x200，放大 3 倍

## 实现标记过滤功能

1. 「标记」菜单下增加二级菜单「过滤」，有四个选项：「所有、有标记、有未标记、无标记」，默认选择「所有」  
2. 当过滤为「所有」时，原图、图1、图2 浏览顺序保持现有逻辑，不过滤图片列表  
3. 当过滤为「有标记」时，三个文件夹的列表过滤出图1 或图2 有标记时对应次序的图片  
4. 当过滤为「有未标记」时，三个文件夹的列表过滤出图1 或图2 有任一未标记时对应次序的图片  
5. 当过滤为「无标记」时，三个文件夹的列表过滤出图1 和图2 均未标记时对应次序的图片  
6. 根据过滤后的列表进行浏览  
7. 仅在菜单中更改过滤条件时刷新列表，浏览过程中改变图1 或图2 的标记状态时不需要动态刷新列表

## 图片列表功能

1. 在「显示-切换遮罩/图片」菜单下增加一个菜单项「图片列表」。点击该菜单后该项被选中  
2. 图片列表是一个非模态的窗口，水平放置一个图片预览列表，最多显示 10 张已打开的图片，支持左右滚动预览更多图片  
3. 图片列表中将当前原图显示的图片进行高亮，在列表中点击某张图片时，主窗口也跳转到这张图片进行联动

## 盲选模式

针对皮卡比（PicPicker）项目的盲选模式：

「盲选模式」是为图1、图2 进行公平盲选标记的功能，避免除图片本身之外的信息（如文件夹信息、文件名）干预标记。

1. 在「显示」-「隐藏图片信息」菜单上方增加菜单「盲选模式」，点击该菜单后该项被选中，表示启动盲选模式  
2. 每次启动盲选模式时，将图1、图2 的显示位置随机对调：即对于部分图片，图1 位置显示的是图2 内容，反之同理。需要进行对调的内容有图片预览框、图片文件名、图片底部文件路径和尺寸信息（包括遮罩）；但文件夹路径、当前标记统计信息保持不对调  
3. 每次启动盲选时随机对调位置的图片数量比例占比可以在 20%～80% 浮动  
4. 盲选模式期间发生的标记以实际选择为准，即标记图1 时，若图1 位置实际显示的是图2，则实际标记数据是图2。已有标记数据不随盲选模式的进入和退出发生变化  
5. 退出盲选模式时，恢复第 2 项描述的对调内容
