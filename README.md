# 比卡拾图 PicPicker

比卡拾图用于最多三个图片文件夹并排对比、逐张同步浏览，适合质检、选图与版本对比。

![PicPicker 预览图](picpicker.png)

## 关于 AI vibe coding

本项目从需求整理、技术选型到功能设计与实现，完整采用 AI 辅助的 vibe coding 方式推进：通过在 `spec.md` 中逐步迭代需求与细节，再由 AI 生成与修订代码，实现了几乎全程「对话驱动开发」。你可以将 `spec.md` 视作本项目的「设计蓝本」，其中记录了功能目标、交互细节以及若干实现思路，代码则是在这一蓝本的基础上由 AI 持续补全与打磨。

## 前置要求

- Python >= 3.11
- [uv](https://github.com/astral-sh/uv)
- [just](https://github.com/casey/just)

## 安装

```bash
just install        # 安装依赖
just install-dev    # 安装开发依赖
```

## 使用

运行 GUI：

```bash
uv run picpicker
# 或
just run
```

主要功能：

- 三栏同步预览（原图 / 图1 / 图2）与上下翻页
- 图1、图2 标记及已选数量/占比统计，支持反转标记和重置标记
- 为图1、图2添加文字备注，并随标记一起导出或从 CSV 恢复
- 导出与从 CSV 导入标记，支持将 CSV 文件直接拖入窗口打开
- 图1、图2 遮罩文件夹绑定与切换
- 按标记导出原图与图1/图2、遮罩为 ZIP
- 放大镜、标记过滤、图片列表小窗、盲选模式与多种图片背景色
- 按文件名搜索原图，以及跳转、对齐
- 将文件夹拖入预览框快速加载，或将当前预览图拖到文件夹中复制源文件

### 新增功能说明

- **备注与备注上屏**：按 `[`、`]` 分别编辑当前图1、图2的备注。备注默认显示在对应预览框底部并自动换行，可按 `C` 隐藏或重新显示；备注会写入 CSV，也会在打开标记文件时恢复。
- **搜索原图**：按 `Cmd+F`（macOS）或 `Ctrl+F`（Windows/Linux）输入文件名关键词，程序会忽略大小写并按“包含关键词”匹配原图。连续搜索同一关键词可循环跳转全部结果；启用标记过滤时，只搜索当前过滤结果。
- **拖拽打开**：可将图片文件夹拖到对应预览框中加载。图1或图2已加载图片时，再拖入文件夹可选择将其作为图片文件夹或遮罩文件夹；也可将 CSV 标记文件直接拖入程序窗口打开。
- **拖拽复制预览图**：从原图、图1或图2预览框向桌面或其他文件夹拖拽，可复制当前实际显示的源图片；显示遮罩或使用盲选模式时，复制内容与预览框当前显示内容一致。
- **标记批量操作**：使用 `Cmd+R` / `Ctrl+R` 反转图1、图2的全部标记，使用 `Cmd+Shift+R` / `Ctrl+Shift+R` 重置全部标记，两项操作都会在执行前确认。
