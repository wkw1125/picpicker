# 比卡拾图 PicPicker

比卡拾图用于最多三个图片文件夹并排对比、逐张同步浏览，适合质检、选图与版本对比。

## 前置要求

- Python >= 3.11
- [uv](https://github.com/astral-sh/uv)
- [just](https://github.com/casey/just)

## 安装

```bash
just install        # 安装依赖
just install-dev    # 安装开发依赖
```

## 使用

运行 GUI：

```bash
uv run picpicker
# 或
just run
```

源码运行时，窗口图标会自动使用项目根目录下的 `logo.png`。

## 打包应用

```bash
just build
```

打包结果位于 `dist/`：在 macOS 上为 `PicPicker.app`，Windows 上为 `PicPicker.exe`，Linux 上为 `PicPicker` 可执行文件。PyInstaller 会将 `logo.png` 转换为当前系统所需的应用图标格式，并将原图一起打包，供程序运行时设置窗口图标。

## 主要功能

- **同步对比**：并排显示原图、图1和图2，支持上下翻页、按序号跳转、对齐原图，以及通过原图列表小窗快速定位。
- **标记与过滤**：分别标记图1、图2并统计已选数量和占比，可按多种标记状态过滤图片。使用 `Cmd+R` / `Ctrl+R` 反转全部标记，使用 `Cmd+Shift+R` / `Ctrl+Shift+R` 重置全部标记，两项操作都会在执行前确认。
- **文字备注**：按 `,`、`.` 分别编辑当前图1、图2的备注。备注默认显示在对应预览框底部并自动换行，可按 `/` 隐藏或重新显示；备注会随标记写入 CSV，并在打开标记文件时恢复。
- **原图搜索**：按 `Cmd+F`（macOS）或 `Ctrl+F`（Windows/Linux）输入文件名关键词，忽略大小写并按包含关系匹配。连续搜索同一关键词可循环跳转全部结果；启用标记过滤时，只搜索当前过滤结果。
- **遮罩对比**：为图1、图2分别绑定遮罩文件夹，并在图片和对应遮罩之间同步切换。
- **辅助查看**：提供三栏同步放大镜、盲选模式、图片信息显隐和多种图片背景色。
- **导入与导出**：将文件夹、文件名、标记、遮罩和备注保存为 CSV，也可从 CSV 恢复工作状态；还可按标记将原图、图1/图2、遮罩和 CSV 打包导出为 ZIP。
- **拖拽操作**：将图片文件夹拖到对应预览框即可加载；图1或图2已有图片时，可选择将拖入文件夹用作图片或遮罩。CSV 标记文件可直接拖入窗口打开。
- **图片右键操作**：可打开当前预览图片、在系统文件管理器中定位该文件、选择指定应用打开、将原图文件拷贝到剪贴板，或仅拷贝文件的绝对路径。
